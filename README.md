# Реализация

### 1. Модулятор 

Класс `QAM_modulate` принимает поток битов и группирует их в символы по 2, 4 или 6 бит в зависимости от типа QAM (4, 16 или 64). Для каждого символа вычисляются вещественная (I) и мнимая (Q) компоненты по формулам из спецификации 3GPP (Specification # 38.883).

  
### 2. Канал AWGN

Класс AWGN добавляет к каждому символу случайный комплексный гауссовский шум с нулевым средним и заданной дисперсией σ². Чем больше дисперсия, тем сильнее размазаны точки на созвездии. Реализована через std::normal_distribution<float> и статический генератор std::mt19937.


### 3. Демодулятор

Класс выполняет **жёсткое решение** (hard decision) с использованием порогов. Для QPSK решение принимается просто по знаку реальной и мнимой части. Для 16‑QAM сначала выполняется нормировка, затем биты определяются по знаку и сравнению с порогом 2. Для 64‑QAM используются три порога (2/√42, 4/√42, 6/√42), последовательное сравнение с которыми даёт три бита для I и три для Q компоненты.

    
### 4. Расчёт BER и сохранение данных

Программа генерирует случайную последовательность бит, модулирует её, а затем прогоняет полученные символы через 21 уровень дисперсии шума — от 0.01 до 1.0. Для каждого уровня добавляется шум, выполняется демодуляция, подсчитывается количество ошибочно восстановленных битов и вычисляется BER (доля ошибок). Результаты сохраняются в CSV‑файл.

### 5. Построение графиков
 
 Отдельный Python‑скрипт читает CSV‑файл с результатами и строит график зависимости BER от дисперсии шума для последнего запущенного типа модуляции. 

В папке Image прикреплены графики, посмотренные с помощью библиотеки DearImGui.
